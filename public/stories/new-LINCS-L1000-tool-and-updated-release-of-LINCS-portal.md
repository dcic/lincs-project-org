## Slicr
The BD2K-LINCS DCIC recently released a new tool called [Slicr](http://amp.pharm.mssm.edu/Slicr) which is a metadata search engine that searches for LINCS L1000 gene expression profiles and signatures matching userâ€™s input parameters. It features download of selected search results as csv files in a zipped folder and visualization of selected results in a 3D scatter plot using PCA or MDS.
